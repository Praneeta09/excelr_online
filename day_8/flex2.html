justify-content have
the property of space-evenly, space-around, space-between, center, flex-start, flex-end

space-evenly will occpy the space start and between and end al will have the space as the same

space-around will have the space around the element and the start and end will have the half of the space

space-between will have the space between the element and the start and end will not have any space

center will have the element in the center and the start and end will not have any space

flex-start will have the element in the start and the end will not have any space

flex-end will have the element in the end and the start will not have any space

flex-wrap will have the element in the next line if the space is not enough

flex-wrap-reverse will have the element in the next line if the space is not enough and it will be in the reverse order

wrap , nowrap we have

flex-direction will have the element in the row and column and row-reverse and column-reverse
